# Black Male College Enrollment Data Analysis

This project analyzes trends in Black male college enrollment using 2023 American Community Survey (ACS) data.

## Tools Used
- Stata
- Statistical Analysis
- Logistic Regression

## Overview
The project examines how income, employment, and demographic factors influence college enrollment among Black men ages 18–25.

## Key Insights
- Income significantly impacts enrollment likelihood
- Full-time employment reduces probability of enrollment
- Black men have lower enrollment rates compared to Black women

## Dataset
American Community Survey (ACS) 2023
