# Black Male College Enrollment Data Analysis

This project analyzes Black male college enrollment using 2023 American Community Survey (ACS) data.

## Tools Used
- Stata
- Survey-weighted analysis
- Logistic regression
- Descriptive statistics

## Overview
This project examines how income, employment, region, and metropolitan status influence college enrollment among Black young adults ages 18–25. It also compares enrollment patterns between Black men and Black women.

## Methods Used
- Data cleaning and variable construction
- Survey weighting
- Cross-tabulations and descriptive statistics
- Logistic regression modeling

## Files Included
- `analysis.do` — Stata code for data cleaning, variable creation, and regression analysis
- `final_paper.pdf` — Full research paper
- `regression_results.xlsx` — Logistic regression output table
- `descriptive_stats.xlsx` — Descriptive statistics table
- `group_comparison.xlsx` — Comparison table for Black men and Black women ages 18–25

## Key Insights
- Black men had lower odds of college enrollment than Black women
- Higher household income was associated with higher odds of enrollment
- Full-time employment was associated with lower odds of enrollment

## Dataset
American Community Survey (ACS) 2023
